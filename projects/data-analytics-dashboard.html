Panel de analítica comercial en Power BI

Solución end-to-end que integra ventas POS, CRM y objetivos trimestrales para brindar seguimiento diario a indicadores clave.

Highlights del dashboard

  • Modelo tabular con más de 120 medidas DAX para márgenes, churn y forecast.
  • Segmentaciones avanzadas por canal, región y familia de producto.
  • Alertas proactivas publicadas mediante Power BI Service + Teams.

Dataset y procesos

El pipeline de datos se construyó con Azure Data Factory y notebooks de Synapse para limpiar información transaccional, enriquecer con datos maestros y publicar el dataset certificado.
