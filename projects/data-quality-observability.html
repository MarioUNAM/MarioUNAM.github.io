Observabilidad de datos 24/7

Implementación de un framework de monitoreo con Great Expectations y Airflow para validar pipelines críticos y publicar alertas en Slack.

Componentes clave

  • Más de 60 suites de pruebas automatizadas para tablas críticas.
  • Metadatos y resultados almacenados en un data lake de Azure.
  • Dashboards de seguimiento en Grafana con métricas de cobertura.

Repo y documentación

El código y la configuración de CI/CD están documentados en un repositorio interno, con ejemplos públicos disponibles para consulta.
