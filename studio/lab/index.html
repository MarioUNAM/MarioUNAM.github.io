Arquetipo 01

Servicio local

Para negocios que viven del barrio: lavanderías, talleres mecánicos, salones de belleza, gimnasios, veterinarias. El sitio empuja una sola acción — pedir el servicio — y reduce las llamadas repetitivas con FAQs y precios visibles.

  • ✓ Hero con propuesta + CTA único (WhatsApp, llamada o reserva)
  • ✓ Mapa con horario y dirección clara
  • ✓ Lista de servicios con precios sin sorpresas
  • ✓ Reseñas Google integradas
Arquetipo 02

Restaurante

Para taquerías, cafeterías, fondas, food trucks. La comida vende sola si la foto está bien tomada; el sitio debe lograr que la gente decida ir antes de googlear "tacos cerca". Carta apetitosa, ubicación al toque, integración con apps de delivery.

  • ✓ Hero foto-driven, menú visual con precios
  • ✓ Botones Uber Eats / Rappi / DiDi en posición prominente
  • ✓ Carrusel de platillos estrella
  • ✓ Reservaciones por WhatsApp con plantillas

Arquetipo 03

Retail

Para boutiques, mercerías, librerías, joyerías, tiendas de barrio con personalidad. La diferencia entre vender o no es la foto y la curaduría. Catálogo navegable, lookbook editorial, checkout por WhatsApp o Mercado Pago.

  • ✓ Catálogo con grid editorial + filtros simples
  • ✓ Fichas de producto con storytelling, no specs
  • ✓ Lookbook estacional como vehículo de marca
  • ✓ Integración Mercado Pago / WhatsApp Business
Arquetipo 04

Profesional independiente

Para dentistas, contadores, abogados, consultores, terapeutas. El cliente está buscando confianza, no precio. El sitio comunica autoridad — credenciales, casos, agenda — sin caer en plantillas de "consultorio dental genérico".

  • ✓ Hero sobrio con propuesta clara + agenda visible
  • ✓ Credenciales y certificaciones documentadas
  • ✓ Calendly / WhatsApp para citas en 1 click
  • ✓ Sección de casos o testimonios con foto real
